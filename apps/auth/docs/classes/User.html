    Class User

    Бизнес-сущность пользователя.

    Инкапсулирует данные и поведение пользователя в системе. Предоставляет удобные методы для доступа к данным, проверки прав и определения статуса.

    Example

    const user = new User(dto);
    console.log(user.displayName); // "Иван Иванов"
    console.log(user.isOnline);    // true
